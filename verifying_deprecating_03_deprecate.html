Deprecate

Deprecate means: this should not be relied on

Deprecation is a stale-content suppression move. It helps reduce noise and confusion when old or misleading content still competes.

Use Deprecate when

The content is outdated, replaced, or actively harmful to the search experience.

Good candidates

  • Superseded process docs
  • Old brand guides
  • Legacy instructions still surfacing

Do not deprecate

  • Current content you just have not reviewed yet
  • Historical content that still has a clear use and is properly labeled
  • Docs that should really be fixed, not hidden
