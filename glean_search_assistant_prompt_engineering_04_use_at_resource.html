Grounding fix

If Glean cites the wrong doc, use @resource

If you already know the right source, reference it directly in the prompt so Assistant grounds on it.

What it solves

The immediate conversation-level problem: wrong or outdated grounding in the current answer.

Scenario

Assistant referenced an outdated internal doc. You know the current page that should drive the answer.

Use the current page with @resource in the prompt.

Example prompt

Use @Current Benefits Guide and summarize the current PTO policy in 5 bullets. If the page does not answer something, say so.
@resource is the fast fix for this answer. It is not the whole fix if the source landscape is messy. Longer term, the source of truth still needs to be current, trusted, and easy to find.
