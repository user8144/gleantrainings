Lesson 3

Cribl scenarios

These are meant to feel like actual work, not sandbox demos.

Scenario 1

Glean referenced an outdated doc

Situation: The answer is plausible, but it leans on an old resource instead of the one you actually need.

Better move: Use @ to point Glean at the current doc, Collection, or known source of truth.

Using @Current Training Source of Truth, summarize the current process and call out any differences from older guidance.

Why this helps: You are not just rephrasing the same prompt; you are correcting the grounding.

Scenario 2

The prompt was too vague

Situation: The answer is generic because the request was generic.

Better move: Add task, context, scope, and output shape.

Summarize the key decisions from our latest internal guidance on this topic for a general employee audience. Keep it to 5 bullets and cite the supporting docs.
Scenario 3

You need the actual source of truth

Situation: A coworker asks, “Where is the current doc for this process?”

Best move: Start with Search. If needed, use Assistant afterward to explain what changed.

Scenario 4

You need a draft from internal context

Situation: You need a first-pass summary or message based on internal material.

Best move: Use Assistant with a clear output shape and ask it to stay grounded in current docs.

Practice prompt

Pick a topic you work with often. Write one prompt that would probably return a generic answer. Then rewrite it with:

a clearer task
a narrower scope
a specific output format
@ grounding if you already know the right source set
