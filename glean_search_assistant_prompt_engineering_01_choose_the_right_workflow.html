Lesson 1

Choose the right workflow

The best result usually comes from choosing the right starting move, not from writing a longer prompt.

Start with Search when…

You need the actual doc, page, deck, or thread Retrieval is the job.
You do not trust the first answer yet Find the source material before asking for synthesis.
You are looking for the current source of truth This is especially important when docs may be old, duplicated, or inconsistent.

Start with Assistant when…

You want synthesis across several sources Summaries, comparisons, and draft-first outputs fit here.
You know the job to be done For example: compare, summarize, extract, draft, or outline.
You still plan to sanity-check the answer Assistant helps you move faster, not skip judgment.

The practical rule

If the deliverable is the source itself, start in Search. If the deliverable is a synthesis of source material, start in Assistant. If Assistant seems off, narrow the source set or route back through Search.

Quick practice

Task: You need the latest version of a team policy and want to send the exact link to a coworker.

Best starting move: Search. The goal is the source, not a paraphrase.

Task: You need a concise summary of three internal docs before a meeting.

Best starting move: Assistant, ideally with clear scope and source guidance.
