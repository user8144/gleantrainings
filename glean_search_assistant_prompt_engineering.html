How to use this module

Each lesson lives on its own page so you can assign, link, or revisit specific skills inside a Glean Collection. The recommended path is linear, but each page also works as a standalone refresher.

Landing 1. Choose the right workflow 2. Prompt lab 3. Cribl scenarios 4. Trust habits 5. Try this this week

What people should leave with

Choose Search vs Assistant more intentionally Know when to retrieve the thing first and when to ask for synthesis.
Write prompts that actually narrow the task Add context, scope, output shape, and source expectations.
Correct weak grounding Use @ to point Glean at the right resource when the first answer leans on an outdated or off-target doc.
Build trust habits Check sources when precision matters and route back to the doc when the doc is the deliverable.
